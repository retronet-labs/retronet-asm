# 01 - Hello 4004

## Obiettivo
Questo esempio introduce il minimo programma osservabile per Intel 4004:
selezione del banco RAM, caricamento di un valore a 4 bit e scrittura sulla
porta di output. E' il corrispettivo didattico di un "hello world".

## Lettura Del Codice
`LDM 0` prepara l'accumulatore per `DCL`, che seleziona il banco RAM 0. Il valore
`5` viene poi caricato nell'accumulatore e mandato alla porta con `WMP`.
L'architettura non ha `HALT`, quindi l'arresto e' simulato con `JUN halt`.

## Stato Atteso
La porta di output del banco selezionato riceve il nibble `5`. Nessuna cella RAM
viene usata come dato persistente.

## Comandi
```sh
go run ./cmd/retronet-asm build examples/i4004/01-hello4004.asm -o hello.rom
go run ../go-4004/cmd/retronet-4004 -trace hello.rom
```
